## 核心实现思路
### 算法原理
采用**回溯法**（试错+回退）求解：
1. 逐行放置皇后，每行仅放1个；
2. 放置前检查当前位置的列、左上→右下对角线、右上→左下对角线是否有皇后；
3. 若当前行无合法位置，回溯到上一行更换列位置；
4. 所有行放置完成则记录一个合法解。

### 关键函数
| 函数名 | 作用 |
|--------|------|
| `solve_n_queens(n)` | 核心求解函数，返回所有合法棋盘布局 |
| `is_valid(row, col)` | 验证(row, col)位置是否可放置皇后 |
| `backtrack(row)` | 递归回溯放置皇后的核心逻辑 |
